# 世界地图模块设计

## 模块目标
- 提供“可阅读、可导航、可移动”的地图探索体验。
- 让玩家在聊天与地图入口之间无缝切换。

## 核心体验规则
- 地图可从聊天区和 Debug 面板双入口打开。
- 地图支持缩放、平移、区块点击查看与移动。
- 移动后地图保持打开，视角状态可保留。

## 参数枚举（设计层）
- 缩放等级：`0%~100%`
- 视角操作：`W`、`A`、`S`、`D`、中键拖动
- 区块节点类型：`zone_node`、`sub_zone_node`、`player_marker`

## 交互约束
- 左侧列表点击区块时先居中再展示详情。
- 当前区块高亮且默认展开子区块。
- 同屏展示区块名称与方位关系，避免纯坐标理解负担。

## 验收标准
- 首次进入可触发地图生成并渲染。
- 重开地图时视角状态可恢复。
- 选择任一区块可进入移动确认流程。
